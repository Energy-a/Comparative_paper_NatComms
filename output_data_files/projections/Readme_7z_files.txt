The .dta files are compressed and saved by country. Each .7z file contains:

second_stage_COUNTRY_db.dta
second_stage_COUNTRY_db_22.dta
second_stage_COUNTRY_wb.dta
second_stage_COUNTRY_wb_22.dta

where the COUNTRY is either 'BRA', 'IND' or MEX'. The data for 'IDN' are not made available to due to purchase rights agreements. 

The .7z files can be uncompressed using 7zip or similar softwares.
